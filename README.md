# Middlefinger AI Live Faceswap

Real-time AI character transformation powered by Middlefinger AI 

## Features
- Real-time video face swapping.
- Credit-based billing system.
- Unique Device ID (Instance ID) tracking.
- WhatsApp integration for credit purchases.

## One-Click Installation (Windows)
1. Download the repository.
2. Double-click `middleInstall.bat`.
3. Follow the prompts. The installer will:
   - Install Node.js and Git if missing.
   - Install all project dependencies.
   - Create a **MiddleFingerAI** shortcut on your Desktop.

## Usage
- Run the app via the Desktop shortcut.
- Enter your Middlefinger API token.
- Upload a reference image.
- Click **Connect & Swap** to start the live stream.

## Requirements
- Node.js (Latest LTS recommended).
- A valid Middlefinger API token.
- A functional camera.
